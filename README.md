# Coffee Shop - Интернет-магазин кофе

Проект создан на основе курса по Next.js с использованием:
- Next.js 14 (App Router)
- TypeScript
- Chakra UI
- Zustand
- Lucide React

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.
